github 사용을 위한 git 사용법

  • 0. 프로젝트 폴더로 이동
  • 1. 저장소 초기화-> git init
  • 2. 현재 폴더의 모든 파일들을 git저장소에 추가 ->git add. (.gitgnore에 선언된 폴더, 파일들은 제외)
  • 3. 커밋, 전송 준비 -> git commit -am 'first'(구별하기 위한 메시지 추가)
  • 4. 브랜치 생성 -> git branch
  • 5. 원격 저장소 지정 -> git remote
  • 6. 푸시, 전송 -> git push 실제 코드를 git hub로 업로드
